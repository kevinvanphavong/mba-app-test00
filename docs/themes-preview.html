Architecture tokens V2 · même HTML/CSS, seul l'attribut data-theme change.

Chaque carte ci-dessus contient exactement le même balisage HTML, recouvert d'un data-theme="…" différent. Aucun composant ne hardcode de couleur — tout passe par les CSS variables (--bg, --surface, --accent…). Les chips teintés sont construits via rgba(var(--raw-X), .15), ce qui leur permet de rester lisibles dans tous les thèmes.
